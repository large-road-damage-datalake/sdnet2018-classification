# Abstract

SDNET2018 Classification is a curated packaging of the SDNET2018 concrete crack dataset for reproducible model development and benchmarking.

The source dataset contains 256x256 RGB image patches from bridge decks, pavements, and walls, labeled as cracked or non-cracked and released with a peer-reviewed data article.

This package preserves publication metadata and provides generated statistics and visualization artifacts for the Large Road Damage DataLake.
